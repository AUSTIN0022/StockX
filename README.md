# StockX

StockX is a web application built using Flask, Python, and the Jinja framework, powered by an SQL database. It leverages the IEX Finance API to provide users with real-time stock data.

## Features

Here's what StockX has to offer:

1. **User Registration and Authentication**: To get started, users need to either register or log in to the platform. This ensures a personalized experience and keeps their data secure.

2. **Virtual Funds**: We provide each user with $10,000 in virtual cash to practice trading without financial risk.

3. **Stock Price Quoting**: StockX enables users to check real-time stock prices by simply entering the stock ticker symbol.

4. **Buy and Sell Stocks**: One of the core functionalities of StockX is the ability to buy and sell stocks. Users can use their virtual funds to purchase shares of companies and sell them when they wish to.

## Installation

To run StockX locally, you'll need the following dependencies:

- Python
- Flask
- Flask-Session
- Requests
- CS50 Library

You can install these dependencies using pip:

pip install flask flask-session requests cs50
